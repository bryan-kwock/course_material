# 英语单词拼图游戏

一个有趣的英语单词拼写练习游戏，答对一题揭开一片拼图。

## 功能特点

- **默认模式**: 使用常用3000词，从易到难，关卡拼图数量为 N² (N=关卡)
- **自定义单词模式**: 上传Excel/CSV文件，使用你自己的单词列表练习
- **错词重复**: 答错的单词会重复出现，直到你答对为止
- **全部掌握**: 所有单词都答对后显示"恭喜掌握了所有单词"
- **生命值**: 5次错误机会，用完游戏结束
- **错词导出**: 游戏结束可以导出错词列表，方便复习

## Excel文件格式

- 第一列：英文单词
- 第二列：中文释义
- 不需要表头

示例文件: `demo_words.csv` (可以直接用Excel打开保存为 .xlsx)

## 使用方法

1. 打开 `index.html`
2. 选择：
   - **使用默认词库开始** - 默认游戏模式
   - **上传Excel文件开始** - 自定义单词学习
3. 根据中文提示拼出英文单词，第一个字母已经给出
4. 填完按 **回车键** 检查
5. 答错扣除一颗心，并提示正确答案，进入下一题（自定义模式错词会重复出现）
6. 答对揭开一片拼图
7. 所有拼图揭开进入下一关
